Python总结

Oct 30 2016

看代码遇到不知道的Python用法的总结.

  • complie()

将source编译为代码，参数为source filename mode，其中mode:exec执行一系列的语句，eval执行一句，single执行单一交互式语句，如print "hello"

code = compile(file.read(),filepath,'exec')

  • exec(code,global_vars,local_vars)

执行存储在字符串或者文件里面的Python语句，执行代码，并且声明全局命名空间和局部命名空间

  • os.environ.get("STRA_ROOT",os.path.join(os.getcwd(),"../.."))

获取环境变量，如果环境变量为空，那么返回第二个参数给的默认值

os.path.join 连接两个字符串作为path

os.getcwd() 获取当前工作目录

  • os.path.normpath(os.path.abspath(root))

首先将所输入的路径转换为绝对路径，然后根据系统环境进行规范化路径

  • map(lambda x:os.path.join(*x),itertools.product(*paths))

itertools.product 表示一个item1，item2等中的项目的笛卡尔积的元组，排列组合

其中*x表示可选数目的参数集合,所以x就是包含所有参数的一个元组，然后将这些元组join起来变成一个路径

  • logging.debug("load config from [%s]",cfile)

后面第二个参数文档里是这样写的 args are the arguments which are merged into msg using the string formatting operator

  • "{:03d}".format(uid % 100)

使用{}形式进行格式化字符串，:表示格式化字符开始，03d表示将数字类型的格式为字符串3位，多处的空用0补齐。

os.path.join(data_root, l1, l2, "{}-{}.{}".format(kind, uid, ext))每一个大括号格式化一个变量

  • datetime变量的使用

date是datetime变量

self._states.get(date,{}).get('stkasset',{})也就是date键存在，则获取数据，否则返回空字典
