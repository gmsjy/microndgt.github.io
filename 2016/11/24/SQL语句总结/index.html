插入

insert into latest_avg_ability (date, ability_risk, ability_gain, ability_short, ability_medium, ability_long) select date, avg(ability_risk), avg(ability_gain), avg(ability_short), avg(ability_medium), avg(ability_long) from summary group by date;

但是group by的属性要是唯一的，这样生成的均值应该是唯一的。

Load by csv

LOAD DATA INFILE '/data/output.csv' INTO TABLE ability CHARACTER SET utf8 FIELDS TERMINATED BY ',';

在load csv的时候出现表中有主键，但是csv数据没有那一列，会报错，doesn't contain data for all columns,于是用Python将每一行的行数加在csv文件的第一列。

这里要注意的是primary_key是从1开始，而不是0.

小操作

删除表

drop table ability

删除表中数据

truncate table ability

修改字段

alter table ability modify column shape_rate varchar(50)

添加索引

alter table ability add index (fundId asc)

选择前10条数据

select * from ability limit 10

添加列

alter table ability add column day_asset double(30,15) null after date;

改变表中字段的顺序

alter table ability_ext fundId fundId int(11) default null after first_day_asset

删除表中某个字段

alter table ability drop column day_asset

修改字段名字

alter table ability change fundid fundId int(11) default null;
