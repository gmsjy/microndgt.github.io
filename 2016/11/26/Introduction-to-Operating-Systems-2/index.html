Simple OS Definition

is a piece of software that abstracts and arbitrates the underlying hardware system

abstract means to simplify what the hardware actually looks like

arbitrate, means to manage to oversee to control the hardware use

Visual Metaphor

An operating system is like a toy shop manager

  • directs operational resources in the shop (control use of employee time, parts, tools…responsible for scheduling for allocating)
  • enforces working policies(fairness, safety, clean-up)
  • mitigates the difficulty of complex tasks(simplifies operation, optimized performance)

What is an Operating System?

Computing systems consist of a number of hardware components

  • one or more processing element,CPUs
  • memory
  • graphics processing cards,GPUs
  • ethernet or wifi cards
  • storage devices like hard disks,SSDs,flash devices like USB drives

Operating System Defination

an OS is a layer of systems software that resides between applications and hardware

  • directly has privileged acess to the underlying hardware
  • hides the hardware complexity
  • manages hardware on behalf of one of more applications according to some predefined policies
  • In addition, it ensures that applications are isolated and protected from one another
