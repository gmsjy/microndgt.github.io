序列倒序

这个问题是我乐此不疲的问题之一，方法有好多种。。。另外，记住拼写，reverse，好多面试者写错了= =

  1. a[::-1]
  2. 对于列表a.reverse(),原地倒序
  3. 对于字符串不可变对象，''.join(list(reversed(a)))

字典排序

这个主要有两种常用方法一种套路

  1. sorted(a.items(), key=lambda x: x[0])
  2. 使用itemgetter
